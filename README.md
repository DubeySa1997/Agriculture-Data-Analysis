# Agriculture-Data-Analysis
Analysis on agriculture data using BI &amp; Cloud Analytics

Excited to Share My End-to-End Cloud Analytics & BI Project! 
Successfully built a complete data pipeline integrating AWS S3 + Snowflake + Power BI Desktop + Power BI Service for cloud-based analytics and reporting.

🔹 Project Workflow:
 ✅ Uploaded CSV data into AWS S3 Bucket using IAM Role-based access
 ✅ Created secure Storage Integration between AWS S3 & Snowflake
 ✅ Loaded and transformed data into Snowflake Data Warehouse
 ✅ Connected Snowflake with Power BI Desktop
 ✅ Built interactive analytical dashboards
 ✅ Published reports to Power BI Service for cloud reporting & accessibility

 ✔ Temperature Analysis
 ✔ Humidity Analysis
